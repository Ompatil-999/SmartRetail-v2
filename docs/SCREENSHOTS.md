## How to Update

When adding new features or updating UI:

1. Take a screenshot of the new feature
2. Name it descriptively (e.g., `Analytics.png`)
3. Save in `Screenshots/` folder
4. Update this file with the new screenshot name

## Best Practices

- Keep screenshots up-to-date with the current UI
- Use consistent sizing and styling
- Include user interactions where relevant (filled forms, populated data)
- Consider adding annotations for complex workflows
